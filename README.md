# Woman Health — UX prototype

Next.js (App Router) mobile-first onboarding prototype with mock data only.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). If the port is busy, run `npx next dev -- -p 3001` and use that URL instead.

## Typography (Stolzl)

Figma uses **Stolzl**. Place licensed `Stolzl-Regular.woff2` and `Stolzl-Medium.woff2` in `public/fonts/` for exact rendering. Until then, **Manrope** is used as a web fallback (Cyrillic-friendly).
